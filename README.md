## Telegram Email Bot

Бот позволяет отправлять электронные письма через интерфейс Telegram, используя простые команды и пошаговые диалоги. Поддерживаются два языка интерфейса: **английский** и **русский**.

### Возможности

- **/start** — показывает личный адрес по умолчанию и краткую информацию.
- **/send** — пошаговая отправка письма:
  - выбор адреса отправителя;
  - ввод списка получателей (до 10 адресов);
  - ввод темы письма (можно пропустить);
  - ввод текста письма и/или прикрепление файлов;
  - подтверждение и отправка.
- **/addresses** — выводит список всех зарегистрированных адресов пользователя.
- **/register** — регистрация нового почтового адреса (указание только левой части адреса).
- **/release** — удаление уже существующего почтового адреса.
- **/help** — выводит справку по доступным командам.
- **/lang** — смена языка интерфейса бота (English / Русский).
- Режим ожидания входящих сообщений с почты с возможностью отмены ожидания.

### Локализация

Локализация реализована в модуле `translator.js`.  
Функция `trans(text, lang)` возвращает текст либо как есть (для `eng`), либо переводит строки интерфейса на русский язык (для `lang != "eng"`). Таким образом, основная логика бота работает с английскими текстами, а пользователю может показываться локализованная версия.

### Требования

- Node.js (рекомендуется актуальная LTS‑версия).
- Зарегистрированный бот в Telegram (через `@BotFather`) и **Bot Token**.
- SMTP‑сервер или почтовый сервис, через который будут отправляться письма (доступ, логин/пароль или OAuth‑данные).

### Установка и запуск

1. Клонируйте или скопируйте проект в локальную директорию:

   ```bash
   git clone <url-репозитория> telegramBot
   cd telegramBot
   ```

2. Установите зависимости (если в проекте есть `package.json`):

   ```bash
   npm install
   ```

3. Создайте файл конфигурации (например, `.env` или `config.json`) и укажите:

   - токен Telegram‑бота;
   - настройки SMTP‑подключения (хост, порт, логин, пароль, адрес отправителя по умолчанию и т. д.).

4. Запустите бота:

   ```bash
   node index.js
   ```

   (Если точка входа другая, используйте соответствующий файл, например `app.js`.)

### Использование

1. Найдите бота в Telegram по указанному имени.
2. Отправьте команду **/start**, чтобы получить адрес по умолчанию и подсказки.
3. Для отправки письма используйте **/send** и следуйте инструкциям бота.
4. При необходимости:
   - добавляйте новые адреса через **/register**;
   - удаляйте ненужные через **/release**;
   - смотрите список адресов через **/addresses**;
   - меняйте язык через **/lang**.

### Структура проекта (примерная)

- `translator.js` — модуль локализации фраз бота (английский ↔ русский).
- `index.js` или `app.js` — основная логика бота: обработка команд Telegram, интеграция с почтовым сервисом, использование `translator.js`.
- дополнительные файлы конфигурации и утилиты для работы с Telegram API и SMTP.
